2.重启nginx

3.MYSQL
python manage.py makemigrations [appname]
python manage.py makemigrations impweb
python manage.py migrate
migrate: 用于执行迁移动作，具有syncdb的功能
makemigrations: 基于当前的model创建新的迁移策略文件
